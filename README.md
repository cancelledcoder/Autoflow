# AutoFlow

AutoFlow is a modern, visual workflow automation platform inspired by n8n and Zapier. Build, automate, and optimize your business or personal processes with an intuitive drag-and-drop interface, AI-powered suggestions, and 50+ integrations.

---

## 🚀 Features

- **Visual Workflow Builder**: Drag-and-drop interface to create complex automations without code.
- **AI Assistant**: GPT-powered suggestions to help you build and optimize workflows.
- **Team Collaboration**: Real-time editing and role-based access for teams (Premium).
- **Workflow Marketplace**: Discover, share, and install community workflows (Premium).
- **Simulation Mode**: Test workflows safely without real API calls (Premium).
- **Advanced Analytics**: Track workflow performance and usage (Premium).
- **Mobile PWA**: Install AutoFlow as an app on your mobile device (Premium).
- **Enterprise Security**: OAuth vault, audit logs, and role-based access (Premium).
- **50+ Integrations**: Connect Gmail, Slack, Google Sheets, Twitter/X, Notion, Webhooks, and more.

## 🛠 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+ recommended)
- [pnpm](https://pnpm.io/) or [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

```bash
git clone https://github.com/cancelledcoder/autoflow.git
cd autoflow
pnpm install
# or
npm install
```

### Running Locally

```bash
pnpm dev
# or
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to use AutoFlow locally.

## 💡 Usage

- **Start Building**: Click "Try Builder" or "Start Building Free" on the landing page.
- **Add Nodes**: Drag services from the sidebar to the canvas to create your workflow.
- **Configure Nodes**: Click on a node to configure its settings.
- **AI Suggestions**: Use the AI Assistant for workflow ideas and optimizations (Premium).
- **Export**: Export your workflow to n8n-compatible format.
- **Feedback**: Use the feedback widget to share suggestions or report bugs.
